🔰 Introduction v3.1

API 3.1 documentation. All requests require an API-Key header or GET parameter. Your API domain, key and personal affiliate key are available in your profile.

🔑 Authorization

curl --header "API-Key:23df424b0a53b0899f78685966243ee61" http://YOUR_API_DOMAIN/
curl http://YOUR_API_DOMAIN/some_path?API-Key=23df424b0a53b0899f78685966243ee61
All API requests must include an API-Key header or API-Key GET parameter.
